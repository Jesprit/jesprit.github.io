Описание команд для работы с SMS-сервисом.
Для получения информации на мобильный телефон, необходимо отправить сообщение на номера…08870000ХХХ
Где ХХХ номера сервисных центров SIB.COM.RU , через №/т центра обслуживания - 7902988990
001 - Сервисы программа TV, прогноз погоды, курс ЦБ РФ.
Ключи для работы с сервисами (только латинские буквы):
TV - что идет по TV, TVZ - что будет идти по TV в ближайшее время
P - прогноз погоды
S - курс ЦБ РФ USD и EUR, AU-Австралийский дол, GB-Английский фунт стерлинг, BE-Белорусский руб, DA-Датских крон, IC-Исландских крон, CA-Канадский дол, NO-Норвежских крон, SY-Сингапурский дол, TU-Турецких лир, UA-Украинских гривен, CH-Швейцарский франк, SW-Шведских крон, JP-Японских иен. HELP - описание всех команд данного сервиса.
003 - Сервисы - центры досуга, кино, театры.
Ключи для работы с сервисами (только латинские буквы):
M - КДЦ Маяковский (кинофильмы) сегодня, MZ - завтра
K - КДЦ Кристалл (кинофильмы) сегодня, KZ - завтра
KO - КДЦ Колизей (кинофильмы) сегодня, KOZ - завтра
P - КДЦ Первомайский (кинофильмы) сегодня, PZ - завтра
S - КДЦ Сатурн (кинофильмы) сегодня, SZ - завтра
HELP - описание всех команд данного сервиса.
100 - Сервис отправки SMS-сообщения на сотовые телефоны Tele2 с городским номером, 'Европа', ФОРА и ФОРА-ПЛЮС.
ХХХХХХ [пробел] сообщение
Номер телефона, на который Вы отправляете текстовое сообщение, должен состоять из шести цифр:
1) Tele2: городской шестизначный номер,
2) 'Европа': шесть последних цифр девятизначного номера,
3) ФОРА: городской шестизначный номер,
4) ФОРА-ПЛЮС: 80[четырёхзначный номер].
101 - Сервис SMS - ICQ-Pager.
Ключи для работы с сервисами:
ХХХХХХ [пробел] сообщение
ХХХХХХ - номер ICQ. Если отправить только номер ICQ то вы получите статус клиента ICQ в сети.
HELP - описание всех команд данного сервиса.
999 - Сервис SMS - CHAT.
Для отправки сообщения с моб/телефона в ЧАТ, необходимо отправить сообщение на номер 999
Ключи для работы с сервисами:
CHAT - Список кто находится в чате и в какой комнате.
