ICQ в Вашем мобильном телефоне!
GIM/GSM Instant Messenger - это мобильный аналог популярного Интернет-пейджера IСQ, который теперь есть в мобильном телефоне абонентов МТС. В любое время абонент сможет общаться с ICQ собеседниками или добавлять новых, даже когда под рукой нет компьютера и доступа в Интернет. Для этого абоненту необходимо быть всего лишь зарегистрированным пользователем на сервере ICQ.
Меню GIM содержит шесть пунктов, с помощью которых пользователь может полностью управлять сервисом: Settings, Status, Contact list, Messages, Search и Services. Settings
Для начала работы абоненту необходимо ввести данные о себе: Nick, UIN и пароль из текущих настроек своей ICQ в меню Settings. Если абонент правильно указал данные, то он получит подтверждение об успешной регистрации в GIM и сразу сможет начать работу с другими пунктами меню.
Status
В меню Status абонент сможет изменять свой статус присутствия в сети. Воспользовавшись командой View, абонент сможет увидеть свой текущий статус в сети, таким, каким его видят другие пользователи в данный момент. Абонент также может изменять свой статус на Online - в сети, Away - отошел, N/A - недоступен, Occupied - занят, DND - не беспокоить, Privacy - видим только для некоторых собеседников, Offline - отключен.
Contact List
Абонент может перенести свой текущий контакт-лист из компьютерной версии ICQ командой Import или создать новый. Увидеть контакт-лист можно с помощью команды View. Воспользовавшись меню Add, абонент сможет добавить имя пользователя в свой контакт-лист. После этого можно переименовать данного собеседника в меню Rename. Для того чтобы удалить имя собеседника из контакт-листа, нужно использовать команду Delete. Если пришел запрос на авторизацию, то в меню Authorization абонент сможет разрешить добавить свое имя в контакт-лист пользователя или игнорировать запрос. Messages
Используя этот пункт меню, абонент может отправить сообщения собеседникам из контакт- листа или любому другому пользователю IСQ, указав UIN.
Search
Поиск собеседника осуществляется в меню Search. Поиск возможен по адресу электронной почты (e-mail) или имени (Nick). Информацию о пользователе ICQ абонент получает, введя UIN.
Services
Дополнительные услуги (дублирование сообщений на e-mail и т.д.) в ближайшее время будут доступны в меню Services.
Стоимость одного исходящего SMS сообщения - 0,11 у.е. (без НДС и НсП)
